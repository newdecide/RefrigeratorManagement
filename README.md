# 냉장고 관리 앱 만들기 

> 만드는 이유

    - 냉장고에 넣는 음식들의 유통기한을 관리하고 버리는 음식이 없도록 하려고 한다.

> 냉장고 관련 앱

    - 우리집 냉장고(1.2 버전) : 수정이 안되서 불편함
    - 우리가족 냉장고(1.2 버전) : 만드려는 앱과 유사함, 유통기한이 2019년 까지만 지원 된다.
    
> 앱 기능 

    - 확장형 리스트뷰(1.보관 분류, 2.물품 내용(날짜, 유통기한))
    - 식품 입력 화면
    - 유통기한 지난 물품
    - 식품 수정 화면
    - 냉장고 데이터베이스 
    - 물품 개수 감소 버튼
